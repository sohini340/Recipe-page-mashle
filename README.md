# Mashel Cream Puff Recipe Website

## Overview
This is a simple, visually appealing website showcasing a detailed recipe for making cream puffs inspired by the anime **"Mashle: Magic And Muscles"**. The page is designed with a warm, vintage Victorian-inspired theme and features step-by-step instructions, ingredients, images, and a styled layout.

The site uses the **UnifrakturCook** Google Font to give a Victorian/authoritative feel, combined with an orange-yellow background for a cozy and inviting appearance.

---

## Features
- **Victorian-style headings** using a unique Google Font.
- **Orange-yellow background** with dark text for readability.
- **Step-by-step recipe instructions** for custard cream and choux pastry.
- **Images** of cream puffs for visual guidance.
- **Styled lists** for ingredients and directions.
- **Copyright footer** included.

---


---

## Technologies Used
- **HTML5** – Structure of the page
- **CSS3** – Styling for fonts, colors, spacing, and images
- **Google Fonts** – 'UnifrakturCook' for Victorian-style headings
- **Inline CSS** – Quick styling for images, text, and sections

---

## How to Use
1. Clone or download the project to your local machine.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
3. Read the recipe and follow the instructions to make cream puffs.

---

## Notes
- The recipe is for **about 8 cream puffs**.
- Images are used for visual reference and can be replaced with your own if desired.
- The website is styled to be simple and readable; additional styling and media queries can be added for full responsiveness.
